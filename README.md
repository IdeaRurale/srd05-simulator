# SRD05 Simulator

Sito statico per il simulatore SRD05 pubblicato con GitHub Pages.

## Pubblicazione

- File principale: `index.html`
- Hosting: GitHub Pages
- Repository: `IdeaRurale/srd05-simulator`
- URL attesa dopo attivazione Pages: `https://idearurale.github.io/srd05-simulator/`

## Attivazione GitHub Pages

1. Apri `https://github.com/IdeaRurale/srd05-simulator`
2. Vai in `Settings` > `Pages`
3. In `Build and deployment`, scegli `Deploy from a branch`
4. Seleziona branch `main`
5. Seleziona cartella `/(root)`
6. Salva

## Web3Forms

Per attivare l'invio email, sostituisci la costante `WEB3FORMS_KEY` in `index.html`
con la tua Access Key Web3Forms associata a `ruggeromanca@gmail.com`.

La chiave va richiesta su `https://web3forms.com/` e viene inviata via email.
